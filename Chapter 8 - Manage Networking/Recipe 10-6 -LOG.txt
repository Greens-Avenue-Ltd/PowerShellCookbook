# 10-6 - Install/authorise DHCP server on DC1 - LOG!


# Step 1


PS C:\> Install-WindowsFeature -Name DHCP `
                   -IncludeManagementTools


Success Restart Needed Exit Code      Feature Result

#  STEP 2

PS C:\> Add-DHCPServerSecurityGroup -Verbose

VERBOSE: Adds DHCP users and DHCP administrators security groups on the DHCP server DC1
